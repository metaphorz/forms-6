FORM S-6 Plan

Goal: Create an interactive grid over the southern part of Florida and run different wind fields over this grid as specified. Details simulations of three types of wind fields have been done in ~/code/catmodel/wind/windfields: the windfield of Powell et al., Holland, and Willoughby. Id also like a 4th option to select called ERA5 if we can work that in. Previous code with ERA5 physical windfields can be found in ~/code/catmodel/wind/era5tracks.

Software: Based on HTML, CSS, and JS with Leaflet used to create a map of the grid specified in Report of Activities in this document.

Report of Activities: Look carefully over these pages in the PDF ROA (Report of Activities): First set of pages is Form S-6 on 182-191 and the grid that is reference on pages 336-341. These all deal with performing sensitivity and uncertainty analysis which is covered in Standard S-3 on pages 167-168. Start with this standard and you will find that this references FORM S-6 as well as the grid, plots, and charts related to sensitivy analysis.
